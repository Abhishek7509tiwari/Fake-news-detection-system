# TruthSeeker: AI Fake News Detector (Gemini Powered)

A modern, production-ready system to analyze and detect fake news using advanced natural language processing powered by the Google Gemini AI.

## 🚀 How to Run

Navigate into the `scripts` folder and double-click one of the corresponding run scripts, or execute them directly from the terminal via the root directory:

### 1. **Streamlit Web Application (Frontend Interface)**
Run `scripts\run_streamlit.bat`
*   Opens the beautiful modern web interface in your browser.
*   Lets you use the default injected API Key directly.
*   Analyze news via an interactive visual UI.

### 2. **Flask Web Server (API Endpoints)**
Run `scripts\run_flask.bat`
*   Starts the Flask backend application (usually on `http://127.0.0.1:5000`).
*   Provides RESTful JSON API `/api/analyze`.
*   Provides an alternative custom-built web page UI at the root `/`.

### 3. **Command Line Interface (Terminal)**
Run `scripts\run_cli.bat`
*   Opens an interactive prompt where you can paste raw news text.
*   Instantly prints Gemini's reasoning and confidence logic without web overhead.

### 4. **Environment Setup**
Run `scripts\setup.bat`
*   If you haven't yet, this cleans and re-establishes your `venv` and installs the required libraries correctly.

## 📁 System Architecture
- `src/core/` - The business logic, AI Pipeline, Web Scrapers, and Alert loggers.
- `src/app_streamlit/` - The Streamlit frontend component files.
- `src/app_flask/` - The Flask frontend templates, CSS, and API router.
- `src/cli/` - The CLI entry point.
- `logs/` - Logs generated by the alert system when Fake News matches signatures.
- `scripts/` - Batch automation files for easy invocation.

*Note: The system now relies entirely on the powerful Gemini 2.5 AI model to perform deep contextual analysis instead of a limited offline trained dataset model.*
