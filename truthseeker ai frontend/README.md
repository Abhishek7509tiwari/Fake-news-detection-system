# TruthSeeker AI — Frontend

A premium, dark-mode frontend for the TruthSeeker AI Fake News Detector. Built with React, TypeScript, Vite, Tailwind CSS, and shadcn/ui.

## Getting Started

```bash
npm install
npm run dev
```

The dev server starts at `http://localhost:8080`. Make sure the Flask backend is running on `http://127.0.0.1:5000`.
